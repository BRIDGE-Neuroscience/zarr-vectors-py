# Ingest formats

Format-specific ingest and export workflows (LAS, PLY, CSV, TRK, TCK,
TRX, SWC, GraphML, OBJ, STL) have moved to the companion package
**`zarr-vectors-tools`**.

This package now provides only the typed core read/write API. See the
companion package documentation for format converters and the CLI.
